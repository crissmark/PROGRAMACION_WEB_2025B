**Proyecto Integrador - Programación Web**

Este proyecto es una aplicación web completa para la gestión escolar, diseñada para facilitar la interacción académica entre Administradores, Maestros y Estudiantes. Permite gestionar materias, inscripciones, entrega de tareas y calificaciones de manera eficiente y segura.

## Características Principales

El sistema cuenta con tres roles de usuario, cada uno con permisos específicos:

### Administrador
- Gestión total de usuarios (CRUD de Maestros y Estudiantes).
- Supervisión global del sistema.

### Maestro
- Creación y gestión de **Materias** (con asignación de grado/cuatrimestre).
- **Sistema de Inscripciones:** Aprobación o rechazo de solicitudes de alumnos.
- Creación de **Tareas** con ponderación específica.
- **Calificaciones:** Revisión de entregas y cálculo automático de promedios.

### Estudiante
- Catálogo de materias disponibles.
- Solicitud de inscripción (Estado: Pendiente -> Aprobado).
- Subida de evidencias/tareas (Archivos PDF, Word, Imagen).
- Consulta de calificaciones y retroalimentación.

## Tecnologías Utilizadas

- **Backend:** PHP 8 (Arquitectura estructurada, PDO para seguridad).
- **Base de Datos:** MySQL (Relacional).
- **Frontend:** HTML5, CSS3 (Estilos personalizados), Bootstrap 5.
- **Scripting:** JavaScript (Validaciones asíncronas).
- **Control de Versiones:** Git & GitLab.
- **Hosting:** Awardspace.
